# Metálica Severo — React + TSX

Estrutura inicial do site em Node.js + React + TypeScript (TSX), inspirada no HTML que você começou.

## Como rodar

```bash
npm install
npm run dev
```

## Como gerar build

```bash
npm run build
npm run preview
```

## O que já vem pronto

- Vite com React + TypeScript
- Componentes por seção
- Conteúdo centralizado em `src/data/siteContent.ts`
- CSS separado em `src/styles/global.css`
- WhatsApp configurado
- Estrutura pronta para receber imagens reais
